# Projet Aircraft

Projet réalisé dans le cadre de la formation Fullstack Data Analysis de Jedha.

## Objectifs du projet Aircraft :

- Utilisation de Snowflake et Deepnote dans le cadre d'une analyse exploratoire des données.
- Construction d'un modèle avec des données nettoyées et structurées à l'aide de DBT.
- Analyse standard pour répondre à des questions sur le secteur aéronautique avec visualisations. 

## Processus

- Entrepôt des données brutes dans Snowflake.
- Construction du modèle en DBT avec des tables staging, dimensions et facts.
- Visualisations effectuées à l'aide de Deepnote.

## Analyse

- Quel avion a effectué le plus de vols ?

- Quel aéroport a transporté le plus de passagers ?

- Quelle a été la meilleure année en termes de passagers-kilomètres payants pour chaque compagnie aérienne ?

- Quelle a été la meilleure année de croissance pour chaque compagnie aérienne ?

## Licence

Projet effectué dans le cadre d'une formation qualifiante, avec des données fictives.
